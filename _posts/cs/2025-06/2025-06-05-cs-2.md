# CS 5장 자료구조
## 📚 5.3 비선형 자료 구조

### 5.3.5 맵 (Map)

#### 📘 정의
- 데이터를 Key-Value 쌍으로 저장하는 자료구조
- Key 를 통해 Value 를 빠르게 조회, 삽입, 삭제할 수 있음

< 특징 >

| 항목        | 설명             |
|-----------|----------------|
| **키** (Key)   | 중복 불가능 (고유함)   |
| **값** (Value) | 중복 허용          |
| 탐색 기준     | 인덱스가 아닌 Key    |
| 내부 구현     | 대부분 해시 테이블 기반  |

---

#### 📌 사용 예시

- 웹 요청 처리: URL ➔ 핸들러 매핑 (Spring Request/Mapping)
- 캐시: Key ➔ Cached Value (ex: Redis, Guava)
- 설정값 관리: `config.get("timeout")`
- 로그 필터링: 사용자 ID ➔ 로그 목록
- 빈 등록: `Map<String, Service>` 으로 구현체 관리

---

#### 🎯 맵 종류

| 언어         | 구현체           | 특징                    |
|------------|---------------|-----------------------|
| JAVA       | `HashMap`       | 순서 없음, 빠른 성능          |
| JAVA       | `LinkedHashMap` | 입력 순서 유지              |
| JAVA       | `TreeMap`       | 키 기준 정렬 (이진 탐색 트리 기반) |
| Python     | `dict`          | 기본 맵 구조, 해시 기반        |
| JavaScript | `Map`           | ES6부터 등장, 키로 객체도 가능   |


---

#### 🏢 시간 복잡도

| 연산              | 시간 복잡도 (Hash 기반) | 설명                        |
|-----------------|------------------|---------------------------|
| `put(key, value)` | 평균 O(1), 최악 O(n) | 요소 삽입, 해시 값 충돌 시 최악 O(n)까지 |
| `get(key)`        | O(1)             | key 로 value 조회            |
| `remove(key)`     | O(1)             | key 로 요소 삭제          |

✚ 해시 충돌
> 서로 다른 Key 가 동일한 해시값을 가질 경우 발생

- 해결 방법:
  - **체이닝**: 같은 버킷에 연결 리스트로 저장
  - **오픈 어드레싱**: 빈 공간 찾기 (선형/이차 탐사 등)

---

#### 💻 Java/Spring 예시

[HashMap 예시]

```java
Map<String, Integer> scores = new HashMap<>();
scores.put("Alice", 90);
scores.put("Bob", 85);

System.out.println(scores.get("Alice")); // 90

  scores.remove("Bob");
System.out.println(scores.containsKey("Bob")); // false
```

#### 🎤 면접 예상 질문
1. Map 의 특징은 무엇인가요?
   - 키-값 쌍으로 데이터를 저장하며, 키를 기반으로 빠르게 조회할 수 있습니다.
   - 키는 중복이 불가능하며, 대부분 해시 테이블로 구현되어 삽입, 조회, 삭제가 평균 O(1) 입니다.
2. HashMap 과 TreeMap 의 차이는?
   - HashMap 은 정렬되지 않은 상태로, 내부 구조는 해시 테이블이며 O(1)의 성능을 갖습니다.
   - 반면, TreeMap 은 키 기준으로 정렬되며, 내부 구조는 BST 기반의 레드블랙 트리로, O(log n)의 성능을 갖습니다.
3. 해시 충돌이 발생하면 어떻게 해결하나요?
   - 해시 충돌 시 체이닝 방식이나 오픈 어드레싱 방식으로 충돌을 해결합니다.
   - 자바의 해시 맵은 기본적으로 체이닝 방식을 사용합니다.
