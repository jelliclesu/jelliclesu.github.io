# 코딩테스트 문제 풀이

## 📘 문제 제목
[백준 - N 과 M (3)](https://www.acmicpc.net/problem/15651)

### 🧠 문제 이해
- 1 부터 N 까지 자연수 중에서 중복 없이 M 개 고른 수열
- 같은 수를 또 고를 수 있음
---

### ✏️ 슈도 코드

```plaintext
i ~ N:
    list.add(i)
    func(count + 1)
    list.remove(count)
```

---

### ✅ 전체 코드
```java
import java.util.*;

public class Main {
  public static int N;
  public static int M;
  public static ArrayList<Integer> list;
  public static StringBuilder sb = new StringBuilder();

  public static void main(String[] args) {
    Scanner sc = new Scanner(System.in);
    N = sc.nextInt();
    M = sc.nextInt();
    list = new ArrayList<>();

    func(0);
    System.out.println(sb.toString());
  }

  private static void func(int count) {
    if (count == M) {
      for (int num : list) {
        sb.append(num).append(" ");
      }
      sb.append("\n");
      return;
    }
    for (int i = 1; i <= N; i++) {
      list.add(i);
      func(count + 1);
      list.remove(count);
    }
  }
}
```
