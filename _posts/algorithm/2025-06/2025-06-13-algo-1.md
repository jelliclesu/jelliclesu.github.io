# 코딩테스트 문제 풀이

## 📘 문제 제목
[백준 - 팩토리얼](https://www.acmicpc.net/problem/10872)

### 🧠 문제 이해
- N! 구하기 = 재귀

---

### ✏️ 슈도 코드

```plaintext
fac(n):
  n <= 1 return
  n * fac(n - 1)
```

---

### ✅ 전체 코드
```java
import java.util.*;

public class Main {
  public static void main(String[] args) {
    Scanner sc = new Scanner(System.in);
    int N = sc.nextInt();

    System.out.println(fac(N));
  }

  private static int fac(int n) {
    if (n <= 1) return 1;
    return n * fac(n - 1);
  }
}
```
