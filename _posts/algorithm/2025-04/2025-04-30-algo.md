# 코딩테스트 문제 풀이

## 📘 문제 제목
[프로그래머스 - H-Index]

### 🧠 문제 이해
- H-Index 구하기: 논문 n편 중, **h번 이상 인용된 논문이 h편 이상**이고 나머지 논문이 h번 이하 인용되었다면 h의 최댓값

---

### ✏️ 슈도 코드

```plaintext
1. 오름차순 정렬
2. 반복:
  h = len - i;
  c[i] >= h ? h
  
  len = 5, i = 0, c[0] = 6 일 때,
  h = 5 < c[0] = 6 이므로 h = 5!
```

---

### ✅ 전체 코드
```java
import java.util.*;

class Solution {
  public int solution(int[] citations) {
    Arrays.sort(citations);
    int len = citations.length;
    int answer = 0;

    for (int i = 0; i < len; i++) {
      int h = len - i;
      if (citations[i] >= h) {
        answer = h;
        return answer;
      }
    }
    return answer;
  }
}
```
