### ✏️ 슈도 코드

```plaintext
1 부터 n - 2 까지 반복:
  sum += i * (n - i - 1);
```

---

### ✅ 전체 코드
```java
import java.util.*;
 
 public class Main {
     public static void main(String[] args) {
         Scanner sc = new Scanner(System.in);
         
         long n = sc.nextLong();
         long sum = 0;
         for (int i = 1; i <= n - 2; i++) {
             sum += i * (n - i - 1);
         }
         System.out.println(sum);
         System.out.println(3);
     }
 }
```
