### ✏️ 슈도 코드

```plaintext
배열로 입력 후 정렬
```

---

### ✅ 전체 코드
```java
import java.util.*;

public class Main {
  public static void main(String[] args) {
    Scanner sc = new Scanner(System.in);
    int n = sc.nextInt();
    int[] arr = new int[n];

    for (int i = 0; i < n; i++) {
      arr[i] = sc.nextInt();
    }
    Arrays.sort(arr);

    StringBuilder sb = new StringBuilder();
    for (int i = 0; i < n; i++) {
      sb.append(arr[i]).append("\n");
    }
    System.out.println(sb.toString());
  }
}
```
- for 문으로 바로 print 하면 시간초과
- `StringBuilder` 활용
