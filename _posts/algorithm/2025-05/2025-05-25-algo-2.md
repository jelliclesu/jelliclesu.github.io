# 코딩테스트 문제 풀이

## 📘 문제 제목
[백준 - 분수 합](https://www.acmicpc.net/problem/1735)

### 🧠 문제 이해
- 두 분수의 합을 기약분수의 형태로 나타내기

---

### ✏️ 슈도 코드

```plaintext
분자 = (a1 * b2) + (a2 * b1)
분모 = b1 * b2

분자와 분모의 최대공약수로 나누기
```

---

### ✅ 전체 코드
```java
import java.util.Scanner;

public class Main {
  public static void main(String[] args) {
    Scanner sc = new Scanner(System.in);
    int a1 = sc.nextInt();
    int b1 = sc.nextInt();
    int a2 = sc.nextInt();
    int b2 = sc.nextInt();

    int ja = (a1 * b2) + (a2 * b1);
    int mo = b1 * b2;

    for (int i = Math.min(mo, ja); i > 0; i--) {
      if (ja % i == 0 && mo % i == 0) {
        System.out.println((ja / i) + " " + (mo / i));
        break;
      }
    }
  }
}
```
