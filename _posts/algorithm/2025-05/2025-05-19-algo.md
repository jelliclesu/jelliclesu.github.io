# 코딩테스트 문제 풀이

## 📘 문제 제목
[백준 - 서로 다른 부분 문자열의 개수](https://www.acmicpc.net/problem/11478)

### 🧠 문제 이해
- 문자열 S 가 주어졌을 때, S 의 서로 다른 부분 문자열의 개수
- 부분 문자열: S 에서 연속된 일부분, 길이가 1보다 크거나 같아야 함
- Set, DP!

✏️ DP   

![img.png](/assets/img/algorithm/2025-05-19.png)

- dp[i][j] = dp[i][j - 1] + dp[j][j]

---

### ✏️ 슈도 코드

```plaintext
dp[i][i] = str[i];

i = 0 부터:
  j = i + 1 부터:
    dp[i][j] = dp[i][j - 1] + dp[j][j]

dp -> set
set.size();
```

---

### ✅ 전체 코드
```java
import java.util.*;

public class Main {
  public static void main(String[] args) {
    Scanner sc = new Scanner(System.in);
    String[] arr = sc.nextLine().split("");
    int len = arr.length;
    
    Set<String> set = new HashSet<>();
    String[][] dp = new String[len][len];
    for (int i = 0; i < len; i++) {
      dp[i][i] = arr[i];
      set.add(dp[i][i]);
    }
    
    for (int i = 0; i < len; i++) {
      for (int j = i + 1; j < len; j++) {
        dp[i][j] = dp[i][j - 1] + dp[j][j];
        set.add(dp[i][j]);
      }
    }
    System.out.println(set.size());
  }
}
```
