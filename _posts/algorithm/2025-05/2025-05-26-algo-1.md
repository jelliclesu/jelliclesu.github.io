# 코딩테스트 문제 풀이

## 📘 문제 제목
[백준 - 가로수](https://www.acmicpc.net/problem/2485)

### 🧠 문제 이해
- 모든 가로수가 같은 간격이 되도록 새로 심어야 하는 가로수의 최소수 구하기
- 기존 나무들 사이에만 새로 심기 가능

---

### ✏️ 슈도 코드

```plaintext
1. 나무 사이 간격 구하기
2. 가장 좁은 간격(min) 찾기
3. 모든 (간격 % min) 이 0 이 될 때까지 min--
4. 모든 (간격 / min - 1) 의 합
```

---

### ✅ 전체 코드
```java
import java.util.*;

public class Main {
  public static void main(String[] args) {
    Scanner sc = new Scanner(System.in);
    int n = sc.nextInt();

    int prev = sc.nextInt();
    List<Integer> list = new ArrayList<>();

    for (int i = 0; i < n - 1; i++) {
      int now = sc.nextInt();
      list.add(now - prev);
      prev = now;
    }

    int min = Collections.min(list);
    while (true) {
      boolean b = true;
      for (int dis : list) {
        if (dis % min != 0) {
          min--;
          b = false;
          break;
        }
      }
      if (b) break;
    }

    int sum = 0;
    for (int dis : list) {
      sum += dis / min - 1;
    }
    System.out.println(sum);
  }
}
```
